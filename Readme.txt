【プレイ人数】1
【入力機器】キーボード と マウス
【フレームレート】60fps


【ゲームプレイ】
キャラクターを操作し、マップのスタートからゴールまで、最速クリアを目指す。

【操作方法】

[マウス]
左クリック：	床に弾丸を発射して、ロケットジャンプする。
マウス移動：	プレイヤーの視角（カメラ）を操作する。


[キーボード]
W、A、S、D: 	4方向に移動
Shift キー：	加速
CTRL キー: 	プレイヤースライド
space キー:　	基礎ジャンプ
E キー:		プレイヤーダッシュ
R キー:		弾丸リロード

【実装機能】
▼ゲーム構成
ゲーム画面

▼プレイヤーの操作設定
